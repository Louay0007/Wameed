## ğŸ› ï¸ How it Works

- **Hero Section**: Combines a countdown timer, animated background, and floating particles for a dynamic intro.
- **Particles**: Custom WebGL/GLSL shaders render interactive, animated particles for a magical effect.
- **Video Player**: Custom controls, smooth transitions, and responsive design.
- **Carousels**: Animated, responsive, and interactive for both countries and partners.
- **Gallery**: Images fade, wiggle, and scale on hover for a lively experience.
- **Audio**: Floating button lets users play/pause background music, randomly selected on each visit.

---

## ğŸ¨ Customization

- **Theming**: Edit `app/globals.css` for colors, gradients, and dark mode.
- **Audio**: Add your own tracks to `public/audio/` and update `components/GlobalAudioPlayer.tsx`.
- **Images & Videos**: Place assets in `public/images/` and `public/videos/`.
- **Animations**: Tweak or add new animations in `app/globals.css` or use `tw-animate-css`.

---
